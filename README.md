# sowpods-six

6-letter words in [SOWPODS](https://en.wikipedia.org/wiki/SOWPODS).

## Example

``` javascript
const sowpodsSix = require('sowpods-six');
// => [
// =>   'aahing',
// =>   'aaliis',
// =>   'aarrgh',
// =>   ...
// =>   'zymoid',
// =>   'zymome',
// =>   'zythum'
// => ]
```

## Installation

``` bash
$ npm install sowpods-six
```

## API

``` javascript
const sowpodsSix = require('sowpods-six');
```

### `sowpodsSix`

An _Array_ of 6-letter words.
